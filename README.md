## 版本说明

本项目会尽量修改明显由于繁简转换造成的错误文字，例如“脚气 -> 香港脚”、“内存 -> 记忆体”等。

关于已修正和即将修正的内容错误，参见 [Issues List](https://github.com/frankslin/daizhigev20/issues?q=is%3Aissue)。
